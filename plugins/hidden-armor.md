# hidden armor

### позволяет игрокам скрыть свою броню чтобы показать свои скины.

использование:

`/togglearmor` - переключить видимость брони

{% hint style="info" %}
кожаная броня не может быть скрыта во избежание проблем с рыхлым снегом
{% endhint %}
